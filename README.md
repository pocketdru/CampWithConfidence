# Camp With Confidence

Search for Campsites in any National park and get locations, and Weather for the park you searched for. Uses Leaflet API, Weather API and the National Parks API

## Built with 

1. Bootstrap.
2. APIs
3. jQuery.
4. AJAX.
5. Firebase.
